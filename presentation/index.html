The Problem

Traditional attendance is slow & error-prone

Schools still rely on paper-based roll calls, manual registers, and spreadsheets — wasting valuable class time and leading to inaccurate records.

AI Features

Intelligence built in

📸
Live Face Detection Webcam feed scanned in real-time using SSD MobileNet — students are recognised automatically
🧬
128-D Face Embeddings Each student's face stored as a vector — compared with Euclidean distance matching
📵
Phone Detection in Exams AI flags when a student's phone is visible during exam sessions — admin gets instant alert
👤
Unknown Face Alerts Any face not in the student database triggers a warning badge in the admin panel
AI Recognition Flow
🎥 Webcam captures frame
↓
🔍 SSD MobileNet detects faces
↓
🧬 FaceNet generates 128-D vector
↓
📏 Euclidean distance comparison
↓
✅ Match found → Attendance marked
🔐 All AI runs 100% in-browser — no face images are ever uploaded to any server
